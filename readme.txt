OneFolio is a one page wordpress theme designed for product presentation purposes or service presentations

== Description ==

A one page wordpress theme having a main product presentation section, a prices section, a services section, a gallery section ,a team section and a contact section. The theme allows you to 
configure each one of these sections in the admin panel using a custom built pagebuilder that simplifies the process of adding ,sorting or removing content.


== Installation ==

1. Upload the entire theme directory to the `/wp-content/themes/` directory
1. Activate the theme through the 'Themes' menu in WordPress
